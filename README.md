## Descripción
    InventarioApp es una aplicación desarrollada en .NET 8 para la gestión de inventarios.  
    Este proyecto incluye:
    - Código fuente en C#
    - Manual de usuario
    - Manual técnico
    - Política de tratamiento de datos personales
    - Terminos y condiciones
    - Instalador 

    El objetivo principal es gestionar el inventario de equipos, dando lugar a funcionalides como:
    - Gestion de Equipos
    - Gestion de Estados
    - Gestion de Categorías
    - Gestion de usuarios
    - Registro de movimientos
    - Registro de historial de movimientos
    - Backups de base de datos
    - Importaciones de backups
    - Descarga de pdf
    - Descarga de excel
    - Visualización de estadisticas del sistema.

## Recomendaciones
    Se recomienda un sistema operativo Windows.
